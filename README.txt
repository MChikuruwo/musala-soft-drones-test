Executing the program: Run instructions;

1. make sure you have jdk 17 or later as the program runs on java 17
2. Have MySQL installed and running  as application auto-generates database & tables on execution.
PS* depending on your version of MySQL you might need to check and ensure that tinyBlob field is longBlob in the file_storage tables
3. Access the API documentation via swagger on http://localhost:8085/swagger-ui.html 

NB* You can post,retrieve and edit dummy data for Drone, medication and dispatch services respectively in that order.
